宝箱の主役画像や縦長背景素材を配置します。主役は1024×1024px、背景は1080×1920pxが目安です。

## v0.9.3 追加可能な主役素材

透過PNGまたはWebP相当、主役は1024×1024px前後、背景は1080×1920px前後を推奨します。

- `gold_sun.png`：黄金祝福の太陽・大発光
- `energy_core.png`：衝撃波のエネルギー核
- `cat_temple_foreground.png`：猫神殿の前景
- `necessary_expense_stamp_cat.png`：必要経費の猫・領収書・判子

配置後、`assets/manifest.json` の `available` にパスを追加します。未配置時は既存SVGへ自動的に戻ります。
